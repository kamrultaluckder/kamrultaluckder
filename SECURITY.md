# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover any security-related issues or vulnerabilities, please do not open a public issue. Instead, report it directly to maintain the safety of the project.

You can report vulnerabilities by reaching out via:
* **GitHub:** Create a private vulnerability report if enabled, or contact me through my profile networks.

I take security seriously and will investigate and respond to any legitimate reports within 48 to 72 hours. Thank you for helping keep this project secure!
